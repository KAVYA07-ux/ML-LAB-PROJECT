# This systematic review synthesizes the most recent neuroimaging procedures and machine learning approaches for the prediction of cognitive abilities in individuals with cognitive impairments. The major mechanisms of cognitive impairment are vascular and neurodegenerative processes. Early diagnosis of cognitive impairment can facilitate timely interventions to mitigate progression. Methods: I systematically searched PubMed, SCOPUS, and Web of Science databases following Preferred Reporting Items for Systematic Reviews and Meta-Analyses (PRISMA) systematic review guidelines. Results: The multimodality framework and machine learning techniques showed potential for predicting the cognitive abilities in individuals with cognitive impairments. Conclusion: Findings of this systematic review identified that the possibility of using neuroimaging data processed by advanced learning algorithms is promising for the prediction of cognitive impairments. I also provided a detailed description of the challenges that researchers are faced along with future research directions. Keywords: Systematic review, cognitive impairment, Prediction, Machine learning.
